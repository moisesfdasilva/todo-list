# Introdução
## Sobre o Projeto
Esta aplicação permite a criação uma lista de tarefas interativa, com objetivo de controlar o andamento das atividades.
## Tecnologias e Ferramentas empregadas
JavaScript;
HTML; e
CSS.
## Status do Projeto
Aplicação em funcionamento.

# Acesso à Aplicação
